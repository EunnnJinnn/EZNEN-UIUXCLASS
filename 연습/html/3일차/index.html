태그의 성질

html 태그는 크게 2종류의 성질로 구분 됩니다.

  • 블럭(block)요소의 태그
  • 인라인(inline)요소의 태그
블럭 요소 성질의 태그의 특징은 주변 정보로부터 독립되어 자신이 속한 별도의 독립 문단(블럭)을 개행(생성)시켜 줍니다 인라인 요소 태그의 특징은 자신이 속한 문단(블럭)내에 다른 인라인 요소의 태그들과 같은 문단을 사용합니다.(개행이 이루어 지지 않습니다.)
